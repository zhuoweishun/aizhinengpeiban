# 植物百科大全 🌱

> 拍照识别植物，一键获取养殖指南

## 项目简介

植物百科大全是一款基于微信小程序的植物识别应用。用户只需拍照或上传图片，即可识别植物品种，并获取详细的养殖指南。

## 功能特性

### 核心功能
- 📷 **拍照识别** - 拍照或上传图片识别植物
- 🔍 **搜索功能** - 搜索植物名称
- 📖 **详细信息** - 名称、科属、养殖指南
- ❤️ **收藏功能** - 收藏喜欢的植物
- 📋 **识别历史** - 查看过往记录

### 特色功能
- 🤖 **AI 识别** - 使用 Qwen-VL 多模态模型
- 💡 **养殖指南** - 光照、浇水、温度、施肥
- ⚠️ **毒性提示** - 标注是否有毒
- 🏷️ **养殖难度** - 简单/中等/困难

## 技术栈

- **前端**: 微信原生小程序
- **后端**: 微信云开发
- **AI**: 通义千问 VL (Qwen-VL)
- **数据**: 微信云数据库
